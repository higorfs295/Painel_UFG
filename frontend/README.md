# Frontend

Criado sobre Vite + React + TypeScript. Instale a base com
`npm create vite@latest . -- --template react-ts` e depois
`npm i @tanstack/react-query react-router-dom zustand`.

Os arquivos em `src/lib` são código real portado do protótipo (parser SIGAA, grafo de
requisitos, somas com teto de 100%); os componentes/páginas são stubs nomeados segundo a
especificação — o protótipo HTML (`painel_engcomp_ufg.html`) serve de referência visual e
de comportamento para cada um.
